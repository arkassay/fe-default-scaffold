fe-default-scaffold
===================

Front-end scaffold using Yeoman

## Getting Started
requires Yo, Grunt, Bower

```shell
npm install -g yo
npm install -g grunt-cli bower
```

## Includes
Bootstrap

LESS

jQuery

respond.min.js

html5shiv.js

Google Closure Linter

IE compatibility mode meta tag

Social Media meta tags

Customized build scripts


## Setup

```shell
npm install
bower install
```
